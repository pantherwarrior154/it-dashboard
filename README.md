# IT Dashboard

A command-line and GUI IT management tool built with Python.

## Features
- System information collector
- Log file parser and report generator
- Device inventory manager
- Network topology visualizer
- Desktop GUI with Tkinter

## Setup
```
python -m venv venv
source venv/bin/activate   # Linux/macOS
venv\Scripts\activate      # Windows
pip install -r requirements.txt
```

## Usage
```
python main.py
```
